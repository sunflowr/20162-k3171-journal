2016-09-14: Status update

What happened yesterday:

Up until now we have tried to use the canvas objec together with javascript in order to draw the devices. During the day an idea to use CSS instead was formed and was later concived. The result so far looks promising and so far seem to help make the code a bit cleaner and easier to maintain.

The code used to fake found bt devices have also been updated, modularized and moved to it’s own javascript file (btdevice.js). This made the main code much cleaner and easier to get an overview of.

We tried changing the background image to one with a checker pattern to see how the perception changed. The feeling for the new background is positive and seem to give the viewer a better feeling of perspective.

With the newly changed code to use CSS for graphics it was also changed so the background also used CSS and a small test to animate the background was added.

Today:

We added a new background with a centered grid instead to experiment how that would change the perception of depth.
