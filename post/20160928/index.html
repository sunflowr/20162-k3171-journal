2016-09-28: Status update

September 28, 2016 · 1 minute read · module2

  • Tags
  • status
  • arduino

Today we’ve worked some more on the prototype.

  • We attached the sensors and vibrators to the two cloth-pieces. (see attached image)
  • Added communication code using the Wire Library for communication between the two Arduino devices.
  • Fixed battery adaptors for the Arduinos.

Notes

  • Social akwardness
  • Positive/negative feelings subjective - e.g. rollercoaster’s amount of scary-fun depends on the user.
  • Self sacrifice
  • Control frame - give user more control (more control good/bad? more control of other good/bad?)
  • Positive: Breaking ice
  • Negative: Making yourself center of attention / making fool of yoursef.

Issues:

  • Having on Arduino power both of the devices seemed to draw too much current so we decided on using batteries for each device - this should hopefully fix the issue as it’s been tested before that one Arduino indeed can power all five vibrators.
  • We’ve been having issues with the lab wires send with the Arduinos - they seem to either be broken from beginning or breaking too easy. We’ve had to replace a few.

Resolved:

  • It seems the touch will be fine siting on the clothes as far as we’ve tested. No false positives.
