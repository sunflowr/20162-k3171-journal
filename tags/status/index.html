Module2 kick-off

September 22, 2016 · 2 minute read

Was feeling a bit better today so we meet up to talk about the project and do some initial plan for how we were going to work. We decided on going with “Social Awkwardness”. Being in a environment when you don’t know what to talk about or how to get some connection to the other people there. Designing something to break the ice through self-sacrifice. We had a bit of brain storming and came up with a few ideas: Go away jeans - detect when offender try to touch users ass and react to it Clapping glove - a glove that creates a synthesized clapping sound when user claps to exaggerate the clapping.
