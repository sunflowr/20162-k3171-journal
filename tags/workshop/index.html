2016-09-08

September 8, 2016 · 1 minute read

Today I worked some on the coding, try to get a simple version of the idea up and running using HTML5 Canvas and Javascript. Currently there’s some issues with scaling making things look aliased.

It’s currently using manually added fake data from firebase as input.

The idea thus far seem to be that a listening device is represented as a ‘circle’. The size of this ‘circle’ is determined by the amount of found devices. We probablt will need to add some form of scale factor determined from averaging the total found devices between all listeners.

Position and possible animation is not decided on yet.
